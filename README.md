# Emberkeep — World Builder

A standalone, zero-dependency isometric world builder for the
[Emberkeep](https://github.com/novativeai/emberkeep-game) dragon game. Paint the
playable floor, non-playable ground, decor and level-blocker clouds on an
infinite iso canvas; calibrate each asset; set per-level camera keyframes; and
swap the merge-item art — then export a `world.json` the game ingests.

**Live:** opens straight onto the current shipped world (`default-world.json`),
so anyone visiting the link sees and can edit the real map. Your edits auto-save
to your browser; use **🌍 Current world** to reload the shipped one, or **Load**
to open your own project file.

Single file — `index.html`. Open it directly or serve the folder statically.

Built with [Claude Code](https://claude.com/claude-code).
